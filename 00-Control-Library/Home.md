# Control Library Home

## Purpose

The Control Library helps technical and functional users upload source material, infer the intended documentation type, and generate a cloned document that follows the right approved template.

## Current Scope

Technical documentation is the only active production scope.

Out of scope for now:

- Functional documentation generation
- Peer review automation
- Sora deck creation
- Compliance evidence packs
- Automated publishing

Track future work in [[04-AI-Operating-System/Skills-Roadmap/Future Skills Registry|Future Skills Registry]].
