# Workspace Context

## Workspace

Control Library inside an Obsidian vault.

## Product Intent

Create a controlled AI-empowered environment where users upload documents and receive cloned documentation outputs based on inferred intent.

## Active Focus

Technical documentation generation from uploaded source documents.

## Current Architecture Assumption

The vault acts as a planning, memory, and control layer. It does not yet define the production application stack.
