# Generation Rules

## Mandatory Rules

- Generate technical documentation only when the source intent supports it.
- Preserve source-grounded facts separately from AI assumptions.
- Use [[02-Technical-Documentation/Templates/Technical Documentation Template|Technical Documentation Template]] for active technical clones.
- Record open questions instead of inventing missing implementation details.
- Keep future skills discoverable but inactive until promoted.

## Output Rules

- Every generated clone must include source inputs, assumptions, risks, open questions, and review checklist.
- Do not hide uncertainty.
- Do not mix functional-documentation-only material into technical implementation sections unless it informs technical context.
