# CreateSkill

Create a new reusable Markdown skill file in `3 skills/`.

When this skill is selected from `Run Skill.bat`, the harness asks the user for the new skill's goal, expected input, output structure, hard rules, and validation checks. It then creates a new `.md` skill file in `3 skills/`.

This utility skill does not process files from `1 inbound/` and does not require Codex CLI.
